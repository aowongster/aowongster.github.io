Git Hooks

If you are a git user, you can write scripts to tie into the git lifecycle.
For example, to publish this blog, I use to a git post-commit hook to automatically
publish and deploy any updates after every commit.

Here’s what it looks like:

This bash script will echo out a message and then run the hexo deploy --generate command after every commit on the repository. How neat is that!

Every git repository has a hidden .git folder for managing the repository internals.

To activate this hook, go into your .git/hooks folder and create a file called post-commit.

Now make it executeable with chmod +x post-commit

Getting started with Hexo

After months of inactivity, I’m rebooting my blog. I’ve migrated from
jekyll to hexo. Both are really similar
in that they are supported on github pages, but for some reason, there’s less friction for me getting setup with hexo.

This blog is my custom domain alistairwong.com .
I host the DNS through cloudflare and have them force https and create and SSL cert which is all free for basic websites.

I configure the repo aowongster.github.io with my domain and host all the static files there.

If you are interested in getting setup with hexo, I followed a great post here.
